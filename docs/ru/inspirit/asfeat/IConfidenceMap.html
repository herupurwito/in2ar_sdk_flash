public function getConfidence(rect:Rectangle, squared:Boolean = false):Number

Get confidence of specified region Higher value means the image most likely not overlayed by anything in this region Smaller value means there is smth above (ex. user hand)

Parameters

rect:Rectangle — what region of image to check
 
squared:Boolean (default = false) — if true return will be squared (skip Math.sqrt call)

Returns
Number — confidence value [0 1] or squared
